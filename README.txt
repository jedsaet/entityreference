Installing Entity Reference Tree:

1. Unpack the Entity Reference Tree module in the appropriate directory of your Drupal installation, most commonly:  sites/all/modules/
2. Enable the Entity Reference Tree module in the administration tools.
3. Add to your taxonomy term reference, entityreference (term only), or OG group widget tree option

Please note that for og_audience, you need to activate first OG Tree Views, available at http://drupal.org/sandbox/lahode/1325062
